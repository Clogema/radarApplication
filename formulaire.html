Configuration Commands :

Each following command is mandatory.

dfeDataOutputMode :

only 1 for 14xx; only 2 or 3 for 16xx.

channelCfg :

15 : for 4 antennas.

0x5 : mask for 2 azimuth antennas; 0x7 : mask for azimuth + elevation antennas.

Not applicable, set to 0

adcCfg :

Number of ADC bits : 0 for 12 bits, 1 for 14 bits, 2 for 16 bits. Only 16 bits are supported for 14xx and 16xx.

Output format : 0 for real, 1 for complex (image band filtered output), 2 for complex (image band visible). Only complex modes are supported for 14xx and 16xx.

adcbufCfg :

Not exist for 14xx, set to -1 for 16xx.

ADCBUF output format : 0 for complex, 1 for real. Only complex mode for 14xx and 16xx.

ADCBUF IQ swap selection : 0 for I in LSB and Q in MSB, 1 for Q in LSB and I in MSB. Only 0 for 14xx and 16xx.

ADCBUF channel interleave config : 0 for interleaved, 1 for non-interleaved. Only 0 for 14xx, only 1 for 16xx.

Chirp threshold configuration used for ADCBUF buffer to trigger ping/pong buffer switch. Valid values : only 1 for 14xx, 0-8 for 16xx.

profileCfg :

Profile Identifier.

Frequency Start in GHz (float).

Idle time in us (float).

ADC valid start time in us (float).

Ramp end time in us (float).

Tx output power back-off code for tx antennas. Only 0 has been tested.

Tx phase shifter for tx antennas. Only 0 has been tested.

Frequency slope for the chirp. Must be greater than 0.

Tx Start Time in us (float).

Number of ADC samples collected during "ADC Sampling Time".

Frequency slope for the chirp. Must be greater than 0.

HPF1 ("High Pass Filter 1") corner. 0 for 175kHz, 1 for 235kHz, 2 for 350kHz, 3 for 700kHz.

HPF2 ("High Pass Filter 2") corner. 0 for 350kHz, 1 for 700kHz, 2 for 1.4MHz, 3 for 2.8MHz.

Rx gain in dB (valid values 24 to 48).

chirpCfg :

Should match with the previous profileID.

Tx antenna enable mask.

1 : Frame trigger delay in ms.
